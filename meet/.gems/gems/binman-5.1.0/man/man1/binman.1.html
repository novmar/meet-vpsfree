BINMAN 1 2016-02-28 5.1.0

NAME

binman - deprecated; use binman-* instead

SYNOPSIS

binman [OPTION]... COMMAND

DESCRIPTION

Runs the fellow "binman-COMMAND" programs listed under "Commands" below.

Note: This program is deprecated for removal in the next major version. To prepare yourself, please run the fellow binman-* programs directly instead.

OPTIONS

-h [PATTERN], --help [PATTERN]
Show this help manual and optionally search for PATTERN regular expression.

COMMANDS

text ...
Runs binman-text(1).
roff ...
Runs binman-roff(1).
html ...
Runs binman-html(1).
show ...
Runs binman-show(1).
help ...
Runs binman-help(1).
