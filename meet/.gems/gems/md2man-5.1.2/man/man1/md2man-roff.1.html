NAME

md2man-roff - convert md2man(5) flavored Markdown text into roff(7)

SYNOPSIS

md2man-roff [OPTION]... [FILE]

DESCRIPTION

This program converts the md2man(5) flavored Markdown text from the given FILE into roff(7) and then prints the result to the standard output stream. If FILE is not given, then the standard input stream is read in its place.

Limitations

This program does not convert the following Redcarpet nodes into roff(7):

  • block_html
  • strikethrough
  • superscript
  • image
  • raw_html

It issues a warning when it encounters these instead. Patches are welcome!

OPTIONS

-h [PATTERN], --help [PATTERN]
Show this help manual and optionally search for PATTERN regular expression.
